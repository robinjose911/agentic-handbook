# Diagram sources

Mermaid sources for the 15 handbook diagrams. Each `NN-name.mmd` here renders to
`../NN-name.png` via the `/mermaid-render` skill (theme per `repo.config.json`). Keeping the sources
in-repo means contributors can edit the diagrams.

> 🚧 The `.mmd` sources are authored in Stage 1. Until a diagram is rendered, its slot in the
> [gallery](../README.md) shows a labeled placeholder so the render-check stays green.
