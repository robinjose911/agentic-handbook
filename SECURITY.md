# Security Policy

> 🚧 Skeleton — expanded in Stage 7.

## Scope

This repository ships **documentation, templates, diagrams, and synthetic reference examples**. The
runnable examples execute **offline against a deterministic mock LLM provider** — no API keys, no
network, no real customer data. There is no deployed service here to attack.

## Reporting a vulnerability

If you find a security issue in the tooling (the preview harness, validators, mock provider, or an
example), please open a private security advisory on the repository rather than a public issue.

## A note on the content

The handbook documents real-world agent security failures (the lethal trifecta, prompt injection,
named incidents) for **defensive, educational** purposes. Every such claim is labeled and cited.
